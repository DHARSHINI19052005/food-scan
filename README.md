# Food Scan

A full-stack web application for food scanning and analysis.

## Project Structure

- **client/**: Frontend React application
- **server/**: Backend Node.js server with API routes and models

## Features

- Scan food items
- View scan history
- User-friendly interface

## Installation

### Prerequisites
- Node.js
- npm or yarn

### Setup
1. Clone the repository:
   ```
   git clone https://github.com/DHARSHINI19052005/food-scan.git
   cd food-scan
   ```

2. Install dependencies for the client:
   ```
   cd client
   npm install
   ```

3. Install dependencies for the server:
   ```
   cd ../server
   npm install
   ```

## Usage

1. Start the server:
   ```
   cd server
   npm start
   ```

2. In a new terminal, start the client:
   ```
   cd client
   npm start
   ```

3. Open your browser to `http://localhost:3000` (or the port specified).
